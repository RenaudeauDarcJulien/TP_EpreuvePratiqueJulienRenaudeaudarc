WEBSERVICE ->
	Comporte 4 fichier:
		- Connexion a la base de donnée de MySQL
		- Un login qui permet de verifier un nom de compte et un mot de passe SELECT
		- Un Register qui permet d'enregistrer un utilisateur avec un INSERT
		- Un Projet qui permet de renvoyer les projets d'un utilisateur via un ID SELECT

APPLICATION ANDROID ->
	Objectif:
		- Réaliser une connexion utilisateur 
		- Acceder au Projet d'un utilisateur
		- Pourra accéder aux notes suivant le projet choisie par l'utilisateur
		- L'utilisateur pourra ajouter des notes avec une categorie précise

	Réalisé:
		- Connexion utilisateur
		- Ajoute Utilisateur
		- Récupération de Projet par ID
		- Affichage de données de la BDD



Mettre dans fichier WebService dans la racine www de WAMP
Si le projet android ne fonctionne pas vérifier les adresses dans les fichiers java
Executer le fichier SQL dans wamp pour créer la base de données
